# houseloan
這是一個基於 C# WinForm 開發的房貸計算工具，專為提供直觀、準確的房屋貸款評估而設計。使用者可以透過簡單的輸入，快速了解每月還款支出、利息總額及各項還款明細。


功能

雙自備款輸入設計：支援以「百分比 (%)」或「固定金額」輸入自備款，系統自動切換邏輯。

寬限期模擬：支援輸入寬限期年限，模擬銀行前幾年「只繳息不還本」的真實狀況。

高精確度計算：採用標準本息平均攤還公式，結果精確至小數點後兩位。

視覺優化介面：

半透明 (ARGB) 排版：結合背景圖與半透明面板，兼顧美觀與易讀性。

流暢操作體驗：支援 Enter 鍵快速計算 與 Tab 鍵順序切換，無需頻繁切換滑鼠。


操作

1.輸入資訊：填寫房屋總價、自備款、年利率、年限與寬限期。

2.執行計算：按下「計算」按鈕或直接敲擊鍵盤 Enter 鍵。

3.查看結果：右側面板將即時顯示貸款總額、每月應繳、首期本息、總利息與總還款額
